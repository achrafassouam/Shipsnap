One tracking system to manage all shipments

Container Tracking system allows you to determine current position of containers on the World Map. The application calculates storage time in ports of transhipment. To find a container on the Map just enter the number and the shipping line.

How does it work ?

The container tracking API allows users to track shipping containers across various carriers and ports by making authenticated requests using an API key. Users send requests to specific endpoints with necessary parameters (e.g., container number), and the API returns structured data, typically in JSON format, detailing the container's current status, location, and other relevant information..
